# GuyTalk Logo Kit

## Core Direction

Use `GuyTalk.` as the primary logo. The blue period is the signature. It should appear anywhere the brand is presented as a wordmark.

## Files

- `guytalk-wordmark.svg` — primary dark-site wordmark.
- `guytalk-wordmark-light.svg` — wordmark for light backgrounds.
- `guytalk-icon.svg` — app icon / favicon direction.
- `guytalk-social-avatar.svg` — profile-photo safe version for Instagram, TikTok, X, YouTube, and Threads.
- `guytalk-g-dot.svg` — capital `G.` concept for profile/icon use.
- `guytalk-gt-lean.svg` — capital `GT` concept with blue leaning `T`.
- `guytalk-gt-lean-transparent.svg` — transparent-background version for placing on graphics.

## Rules

- Brand name in copy: `GuyTalk`, not `GuyTalk Media`.
- URL / handles: `guytalkmedia.com`, `@guytalkmedia`.
- Primary colors: navy `#0B1322`, white `#F2F4F7`, blue `#3D7EFF`.
- Keep the logo flat. No glow, bevel, mascot, or busy mark.
- For tiny contexts, use either the `G.` mark or the `GT` mark instead of the full wordmark.
